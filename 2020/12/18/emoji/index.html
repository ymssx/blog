让emoji固定风格

2020年12月19日 12:36 # 陕西-西安
javascript emoji

# emoji

在文档中适当的加入一些emoji可以让文字变得活泼生动起来，然而emoji在不同的终端下有着不一样的样式，这让网页的美观性变得不可控。 我更喜欢微软的扁平化emoji，风格简洁而且现代化


所以如何让emoji在其它终端（如安卓、苹果）也保持这个风格呢?

# 提取资源

一个自然的想法是将所有的emoji以图片存储起来，然后在网页中用图片代替emoji。

我选择在网上用爬虫爬取现有的资源

emojipedia https://emojipedia.org/

# 简单封装

将<img>封装在<i>标签内，然后获取其dom节点，通过getComputedStyle方法获取所在文档流的font-size，然后自动将emoji图片的宽高设置为字体高度即可。

同时可以判断当前操作系统是否为Windows，如果是则可以直接显示emoji，从而节省图片流量。否则使用图片的兼容手段。

下面是我在vue中的代码实践 

<template>
  <i class="emoji" ref="font">
    <img v-if="!isMs" :src="`/emoji/120/${i}.png`" :style="{ width: 'auto', height: size }" />
    <span v-else>{{emoji}}</span>
  </i>
</template>

<script>
import emojiMap from './emoji';

export default {
  props: ['i'],
  data() {
    return {
      size: '1rem',
    };
  },
  computed: {
    emoji() {
      return emojiMap[this.i];
    },
  },
  methods: {
    isMs() {
      // 判断是否是Windows操作系统
      const agent = navigator.userAgent.toLowerCase();
      return (
        agent.indexOf("win32") >= 0 || agent.indexOf("wow32") >= 0 ||
        agent.indexOf("win64") >= 0 || agent.indexOf("wow64") >= 0
      );
    },
  },
  mounted() {
    const height = window.getComputedStyle(this.$refs.font).fontSize;
    this.size = parseFloat(height) * 1.1 + 'px';
  }
}
</script>

<style lang="scss">
.emoji {
  font-style: normal;
  
  img {
    vertical-align: text-bottom;
  }
}
</style>

# 爬虫代码

我选择使用Node.js上的puppteer库进行爬虫

const fs = require('fs');
const puppeteer = require('puppeteer');
const Bagpipe = require('bagpipe');
const request = require('request');

const downloadImage = (src, dest, callback) => {
	request.head(src, (err, res, body) => {
		if (src) {
			request(src).pipe(fs.createWriteStream(dest)).on('close', () => {
				callback(null, dest);
			});
		}
	});
};

const getEmojis = async (url) => {
  const browser = await puppeteer.launch({
    ignoreHTTPSErrors: true,
    headless: false,
  });

  const page = await browser.newPage();

  await page.goto(url, { waitUntil: 'domcontentloaded' });

  const grid = await page.$('.emoji-grid');
  const list = await grid.$$eval('li', nodes => nodes.map(n => n.innerHTML));

  const bagpipe = new Bagpipe(100);

  const emojiMap = list.forEach((item) => {
    const url = item.match(/https:\/\/emojipedia-us.s3.dualstack.us-west-1.amazonaws.com\/thumbs\/144\/microsoft.*?\.png/)[0];
    const name = item.match(/(?<=\<a href="\/).*?(?=\/\")/)[0];
    const savePath = `./emoji/${120}/${name}.png`;

    bagpipe.push(downloadImage, url, savePath, (err, data) => {});
  });
};

getEmojis('https://emojipedia.org/microsoft/');
